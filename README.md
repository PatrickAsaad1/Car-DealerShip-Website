## 🎯 Overview

A modern, fully responsive car dealership website featuring a complete user authentication system including registration, login, and password recovery functionality.

The site showcases a premium automotive aesthetic with dark glass-morphism design, custom backgrounds, and intuitive user flows.

---

## ✨ Features

### 🚗 Core Features

- Fully responsive design
- Premium automotive-themed UI
- Glass-morphism design language
- Mobile-first approach

### 🔐 Authentication System

- User registration (Sign Up)
- User login (Login)
- Password recovery flow
- OTP verification (demo)
- Password change functionality

### 🎨 UI/UX Features

- Password visibility toggle
- Real-time validation feedback
- Color-coded input indicators
- Smooth hover animations
- Intuitive navigation between pages

## 📄 Pages

| Page            | Description                                    |
| --------------- | ---------------------------------------------- |
| **Login**       | User authentication with username & password   |
| **Signup**      | New user registration with validation          |
| **Send OTP**    | Password reset request with email verification |
| **OTP**         | 6-digit code verification                      |
| **Change Pass** | Password update with confirmation              |
